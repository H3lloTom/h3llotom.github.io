数组方法

==判断是否是数组方法：==

  • value instanseof Array 多框架之间创建数组，传递数组后，具有不同的构造函数
  • Array.isArray(value) ECMAScript5引入

==转换方法==

  • Array.valueOf() 返回 数组
  • Array.toString 返回用逗号隔开的数组元素组合成的字符串 如“1,2,3”
  • Array.join() 返回使用传入值隔开的数组元素组合成的字符串 （缺省值情况使用逗号）
